PERO actualmente NO hay validación automática. Necesito que implementes esta lógica:

📌 REGLA DEL CAMPO "OTRO":

Cuando un usuario escribe una industria en "OTRO", el sistema debe:

1️⃣ VERIFICAR si esa industria YA EXISTE en la lista maestra pero el usuario no la vio.
   - Si existe → mostrar un mensaje: "Esta industria ya existe como 'X'. ¿Quieres seleccionarla?"
   - Si el usuario confirma → seleccionar automáticamente la existente.

2️⃣ VERIFICAR si la industria NO EXISTE en la lista maestra.
   - Consultar una fuente externa (API, base de datos de industrias reales, o IA) para validar si es una industria REAL.

   - Ejemplo de inventadas: "SuperPoderes Digitales", "Magia con Código".

3️⃣ SI la industria es REAL y NO existe en nuestra lista:
   → AGREGARLA AUTOMÁTICAMENTE a la lista maestra de industrias.
   → PROPAGARLA a TODOS los formularios (registro, creación de negocio, etc.).
   → Mostrar mensaje: "Nueva industria detectada y agregada correctamente: 'X'".

4️⃣ SI la industria es INVENTADA (no real):
   → NO agregarla a la lista maestra.
   → Mostrar mensaje: "No pudimos validar esta industria. Por favor, selecciona una de la lista o escribe una industria real."
