🐛 PROBLEMA ADICIONAL DETECTADO:

El botón o opción "TODOS" en las SUBCATEGORÍAS (subindustrias) NO SE VE o NO EXISTE.

📍 DÓNDE DEBE ESTAR:
Dentro de cada industria, en la sección de subindustrias, debe haber una opción que diga "TODOS" o "Seleccionar todas".

🎯 QUÉ DEBE HACER EL BOTÓN "TODOS":
✅ Selecciona AUTOMÁTICAMENTE todas las subindustrias disponibles.
✅ El usuario no tiene que marcar una por una.
✅ Si luego desmarca UNA subindustria, "TODOS" debe DESMARCARSE automáticamente.

📌 EJEMPLO CONCRETO:

Industria: "Energía Solar"
Subindustrias:
   ☐ Instalación industrial
   ☐ Instalación agrícola
   ☐ Instalación en minigranjas
   ☐ Instalación residencial
   ☐ Instalación de cargadores para carros eléctricos

✅ DEBE HABER una opción adicional:
   ☑ [TODOS] → Al marcarlo, se seleccionan TODAS las de arriba.

📌 CASO DE USO:
Un negocio como "Eco" hace 3 de las 4 opciones (todas menos residencial).

Con el botón "TODOS":
1. El usuario marca [TODOS].
2. Se seleccionan todas.
3. El usuario DESMARCA "Instalación residencial".
4. [TODOS] se DESMARCA automáticamente.
5. Resultado final: 3 seleccionadas (las que quería).

✅ ESTO AHORRA TIEMPO Y MEJORA LA EXPERIENCIA.

🔧 ACCIÓN REQUERIDA PARA EL BOTÓN "TODOS":

1. AGREGAR el botón o checkbox "TODOS" en la sección de subindustrias.
2. IMPLEMENTAR la lógica de selección múltiple:
   - "TODOS" selecciona todas.
   - Si se desmarca UNA subindustria, "TODOS" se desmarca solo.
   - Si el usuario marca TODAS una por una, "TODOS" se marca automáticamente.
3. APLICAR esto para TODAS las industrias que tengan subindustrias.
