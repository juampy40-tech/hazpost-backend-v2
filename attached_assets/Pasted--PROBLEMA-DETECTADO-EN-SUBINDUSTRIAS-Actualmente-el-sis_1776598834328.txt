⚠️ PROBLEMA DETECTADO EN SUBINDUSTRIAS:

Actualmente el sistema tiene un problema con las SUBINDUSTRIAS:

1. No hay campo "OTRO" para subindustrias (solo existe para industrias).
2. No se puede seleccionar MÚLTIPLES subindustrias (solo una).
3. No hay opción "TODOS" para seleccionar todas las subindustrias de una vez.
4. Cuando un usuario escribe una subindustria nueva en "OTRO", no se valida ni se agrega automáticamente.

---

🎯 ESTRUCTURA REQUERIDA:

Industria
   └── Subindustrias
         ├── Opción 1
         ├── Opción 2
         ├── Opción 3
         ├── Opción "TODOS" (selecciona todas automáticamente)
         ├── Opción "OTRO" (campo de texto para escribir)
         └── Botón "Agregar otra subindustria" (para selección múltiple)

---

📌 CASO DE EJEMPLO:

Industria: "Energía Solar"

Subindustrias actuales:
- Instalación residencial
- Instalación comercial
- mantenimiento solar
- asesoría energetica

Un negocio como "Eco" hace 3 de las 4 opciones (todas menos asesoría energetica).

PROBLEMA ACTUAL:
El sistema OBLIGA a seleccionar SOLO UNA subindustria.

LO QUE DEBE PASAR:
✅ El usuario debe poder seleccionar VARIAS subindustrias.
✅ O usar "TODOS" para seleccionarlas todas.
✅ O usar "OTRO" si falta alguna.
✅ O usar "Agregar otra subindustria" para añadir más manualmente.

🔧 SOLUCIONES ESPECÍFICAS REQUERIDAS:

1. CREAR CAMPO "OTRO" PARA SUBINDUSTRIAS
   - Igual que en industrias.
   - Con validación automática.

2. IMPLEMENTAR SELECCIÓN MÚLTIPLE
   - Checkboxes en lugar de radio buttons o select único.
   - Opción "TODOS" que selecciona todas automáticamente.
   - Botón "Agregar otra subindustria" para añadir más opciones sobre la marcha.

3. VALIDACIÓN AUTOMÁTICA DE SUBINDUSTRIAS EN "OTRO"
   Cuando un usuario escribe una subindustria en "OTRO":
   
   ✅ PASO 1: Verificar si ya existe en la lista maestra de subindustrias.
      - Si existe → sugerir seleccionarla.
   
   ✅ PASO 2: Verificar si es una subindustria REAL.
      - Usar API, base de datos o IA para validar.
      - Ejemplo real: "Instalación industrial en techos"
      - Ejemplo inventado: "Energía mágica solar"
   
   ✅ PASO 3: Si es REAL y NO existe:
      - AGREGAR a la lista maestra de subindustrias.
      - PROPAGAR a todos los formularios.
      - ENVIAR a la IA.
      - Disponible para FUTUROS USUARIOS.
   
   ✅ PASO 4: Si es INVENTADA:
      - NO agregar.
      - Mostrar mensaje: "Subindustria no válida. Selecciona una de la lista."

4. AGREGAR SUBINDUSTRIAS FALTANTES (caso concreto)

   Para la industria "Energía Solar", las subindustrias que faltan son:
   
   ✅ Instalación industrial
   ✅ Instalación agrícola
   ✅ Instalación de minigranjas
   
   Estas deben agregarse a la lista maestra AHORA MISMO.

5. OPCIÓN "AGREGAR OTRA SUBINDUSTRIA"
   - Botón dinámico que permite añadir más subindustrias.
   - Cada nueva subindustria debe pasar por la validación.
   - El usuario puede seleccionar múltiples subindustrias combinando:
     * Opciones existentes (checkboxes)
     * Opciones nuevas (campo "OTRO" validado)

---

🧠 LÓGICA DE SELECCIÓN MÚLTIPLE CON "TODOS":

Si el usuario selecciona "TODOS":
→ Automáticamente se seleccionan TODAS las subindustrias disponibles.
→ Si luego desmarca una, "TODOS" se desmarca automáticamente.
